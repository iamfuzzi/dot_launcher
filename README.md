





## Описание групп классов
> класс Main - основной класс запуска приложения

> minecraft - запуск игры и скачивание нужных файлов для ее работы
> text - методы консольного управления лаунчером

> util/general - Вспомогательные методы. Более широкие методы для реализации других методов. Не относятся конкретно к проекту
> util - Вспомогательные методы. Относятся к проекту. Имеют в себе конкретные переменные, значения...



> Временно отсутствуют:
> frames - классы фреймов (окон)
> graphics - классы создания и изменения параметров или объектов в окнах
