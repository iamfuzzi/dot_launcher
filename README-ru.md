> ## Инструкция по установке и использованию
> - Скачать весь исходный код (Главная страница -> Code -> Download ZIP)
> - Распаковать архив в любое удобное место
> - Открыть run.bat*
> - Далее, см. раздел [консольных комманд](#консольные-комманды).


> ## Консольные комманды
> Данный раздел создан для того, чтобы ознакомиться с тем, как использовать лаунчер через консоль (сейчас это единственный вариант, но скоро будет доступен нормальный интерфейс)
> > ### Список консольных комманд
> > - debug - отображение информации из настроек лаунчера, рабочих путей, методов. Нужен для поиска неисправностей.
> > - exit / stop - выход из программы.
> > - jdk - устанавливает Java. Занимает от 30 секунд до нескольких минут.
> > - download <версия> as <имя> - скачивает нужную версию Minecraft и сохраняет ее под нужным именем (имя - название папки и .json, .jar файлов в ней). В основном, занимает не более 20 секунд.
> > - libraries <имя> - скачивает все нужные библиотеки для игры. Около 1 минуты.
> > - index <имя> - скачивает .json файл индекса ассетов для сборки. Не более 5 секунд.
> > - assets <имя> - скачивает нужные ассеты для версии (иконка, панорама главного меню, все звуки и пр.). Может выполняться до 10 минут.
> > - launch <имя> - запускает нужную сборку.
> ## Что делать в случае, если программа не работает или работает плохо?
> Если Вы столкнулись с какой-либо ошибкой, сначала попробуйте все альтернативные загрузщики.
> > ### Изестные проблемы
> > - На Windows 11 может быть проблема с компиляцией программы (проблема с символом 0x98). Решение: (не проверено) использовать run_utf
> 
> > ### Незначительные ошибки
> > - Если Вы решили запустить debug версию лоадера и увидели примерно 5 ошибок, в которых написано что-то вида "URL url = new URL", то не стоит переживать по этому поводу, это никак не влияет на работу программы.
