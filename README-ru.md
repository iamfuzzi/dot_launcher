> ## Инструкция по установке и использованию
> Первым делом, нужно убедиться, что у вас установлена Java. Рекомендуется устанавливать последнюю версию. Это нужно для того, чтобы сразу компилировать программу, т.к. скачивая архив вы скачиваете исходный код, а .bat файлы для запуска сразу компилируют программу.
> 
> Убедившись, что у вас есть Java, начнем установку
> 
> - Скачать весь исходный код (Главная страница (Code) > Code > Download ZIP)
> - Распаковать архив в любое удобное место
> - Открыть run.bat[*](#что-делать-в-случае-если-программа-не-работает-или-работает-плохо)
> 
> 
> 
> Если вы хотите использовать программу через консоль (пока что это единственный вариант), то следует написать эти комманды для создания первой сборки:
> ```jshelllanguage
>   jdk 
>   download -1.21.1 -"Новая сборка" -vanilla -null
>   libraries -"Новая сборка"
>   assets -"Новая сборка"
>   index -"Новая сборка"
>   launch -"Новая сборка"
> ```
> Для дальнейших действий в консоли см. раздел [консольных команд](#консольные-команды).

> ## Консольные команды
> Данный раздел создан для того, чтобы ознакомиться с тем, как использовать лаунчер через консоль (сейчас это единственный вариант, но скоро будет доступен нормальный интерфейс).
>
> Для начала нужно разобраться как запускать какие-либо команды.\
> Любая команда выглядит примерно так:
> 
> `название -аргумент1 -"аргумент номер 2"...`
> 
> Аргументы обязательно передаются с символом **-**. Аргумент можно передавать как просто без кавычек, так и с ними, если нужно передать аргумент, в котором более 1 слова.
> > ### Список консольных команд
> > - debug - отображение информации из настроек лаунчера, рабочих путей, методов. Нужен для поиска неисправностей.
> > - [exit, stop, quit, close] - выход из программы.
> > - jdk - устанавливает Java. Может занимать от 10 секунд до нескольких минут.
> > - download -version -name -loader -loaderversion
> >  - version - версия игры
> >  - name - название, под которым будет сохранена сборка
> >  - loader - загрузщик модов: vanilla, fabric
> >  - loaderversion - версия загрузщика модов. Любой набор букв, если vanilla
> > - libraries -name - скачивает все нужные библиотеки для нужной версии игры. Около 1 минуты.
> > - index -name - скачивает .json файл индекса ассетов для сборки. Не более 5 секунд.
> > - assets -name - скачивает нужные ассеты для версии (иконка, панорама главного меню, все звуки и пр.). Может выполняться до 10 минут.
> > - [launch, run, play] -name - запускает нужную сборку.

> ## Что делать в случае, если программа не работает или работает плохо?
> Если Вы столкнулись с какой-либо ошибкой, сначала попробуйте все альтернативные загрузщики.
> > ### Изестные проблемы
> > - На Windows 11 может быть проблема с компиляцией программы (проблема с символом 0x98). Решение: (не проверено) использовать run_utf
> 
> > ### Незначительные ошибки
> > - Если Вы решили запустить debug-версию лоадера и увидели примерно 5 ошибок, в которых написано что-то подобное: "URL url = new URL", то не стоит переживать по этому поводу, это никак не влияет на работу программы.
